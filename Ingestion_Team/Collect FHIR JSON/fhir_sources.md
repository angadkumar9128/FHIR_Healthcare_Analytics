# FHIR Data Sources Reference

The FHIR JSON files used in this project are collected from open and
industry-recognized sources:

1. HL7 FHIR Sample Resources
   - Official reference implementation
   - Used for understanding standard FHIR schemas

2. Synthea Synthetic Health Data Generator
   - Generates realistic but synthetic patient data
   - Commonly used for healthcare analytics demos and POCs

These datasets are widely accepted for learning, POCs, and enterprise demos.

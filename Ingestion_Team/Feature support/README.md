## Objective
The Silver layer provides normalized, feature-ready datasets derived from
FHIR JSON resources to support analytics, reporting, and downstream Gold
aggregations.

Each Silver table represents a single FHIR resource with clean schema,
stable identifiers, and joinable keys.

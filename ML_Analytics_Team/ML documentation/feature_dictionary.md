# Feature Dictionary

## Feature Scores

| Feature Name | Description | Range |
|-------------|-------------|-------|
| utilization_score | Healthcare usage intensity | 0–100 |
| chronic_burden_score | Chronic condition impact | 0–100 |
| cost_intensity_score | Financial utilization | 0–100 |
| readmission_score | Readmission risk signal | 0 or 100 |

---

## Feature Characteristics

- Patient-level
- Time-window based
- Explainable and auditable

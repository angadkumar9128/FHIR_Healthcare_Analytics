# ML Analytics – Risk Logic Build

## Objective
This module defines the logic used to assess patient risk by combining
multiple feature scores into a single interpretable risk outcome.

Risk logic is designed to be:
- Explainable
- Deterministic
- Clinically interpretable
- ML-ready for future models

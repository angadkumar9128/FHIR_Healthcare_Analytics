## Objective
This module defines healthcare metrics derived from Gold FHIR datasets.
These metrics act as the foundation for feature engineering, risk logic,

## Design Principles

- Metrics are patient-centric
- Time-window based (30/60/90 days)
- Fully explainable
- Derived only from Gold datasets
- Suitable for both analytics and ML models

---

## Metric Categories

1. Utilization Metrics
2. Clinical Metrics
3. Financial Metrics
4. Outcome & Risk Indicators

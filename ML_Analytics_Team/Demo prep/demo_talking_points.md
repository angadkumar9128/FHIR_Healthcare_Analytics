# Demo Talking Points

## 1. Context
"We built a healthcare analytics platform using FHIR data and extended it
with an explainable ML Analytics layer."

## 2. Why ML Analytics
"Raw analytics show what happened. ML Analytics explains risk and priority."

## 3. Explainability
"Every risk score is broken down into feature-level contributions."

## 4. Trust
"No black-box models. Everything is auditable and versioned."

## 5. Scalability
"The design supports future ML models without pipeline redesign."
